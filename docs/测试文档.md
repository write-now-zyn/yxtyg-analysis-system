## 3. 接口测试

### 3.1 接口测试清单
| 接口 | 方法 | 测试要点 |
|------|------|---------|
| /api/experimenter/list | GET | 分页参数、搜索条件 |
| /api/experimenter/add | POST | 必填字段校验 |
| /api/experimenter/update | PUT | 数据更新 |
| /api/experimenter/delete/{id} | DELETE | 删除确认 |
| /api/workorder/import | POST | 文件格式、去重 |
| /api/workorder/list | GET | 分页、搜索 |
| /api/workorder/detail/{id} | GET | 详情查询、解决方案历史 |
| /api/workorder/{id} | DELETE | 删除确认 |
| /api/workorder/batchDelete | POST | 批量删除 |
| /api/review/add | POST | 与会人JSON格式 |
| /api/training/report/add | POST | 培训记录数组 |
