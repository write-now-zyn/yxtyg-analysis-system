# 整体看板功能需求文档

## 1. 功能概述

整体看板功能用于展示各地市每月三个专项（需求提单、评审参与、培训上报）的参与情况和达标情况，帮助管理层直观了解各地市的工作完成状态。

## 2. 业务需求

### 2.1 三项指标达标标准

| 专项 | 达标标准 | 说明 |
|------|---------|------|
| 需求提单 | ≥6次/月 | 每月需求提单数量 |
| 评审参与 | ≥2次/月 | 每月参与评审会议次数 |
| 培训上报 | ≥8次/月 | 每月培训上报次数 |

### 2.2 功能要求

1. **月份切换**：默认显示当前月数据，支持切换查看历史月份
2. **总览展示**：三个专项的总览卡片，显示达标情况和未达标地市
3. **排行展示**：分类展示各地市当月排行（按数量降序）
4. **未达标提示**：未达标地市使用红色字体标注

## 3. 数据来源

| 专项 | 数据表 | 筛选条件 |
|------|--------|---------|
| 需求提单 | t_work_order | order_no前6位匹配月份 |
| 评审参与 | t_review_meeting | meeting_time匹配月份，解析participants JSON |
| 培训上报 | t_training_report | report_month匹配月份 |

## 5. 验收标准

1. 页面正常加载，默认显示当前月数据
2. 月份切换后数据正确刷新
3. 三个总览卡片正确显示达标数量和未达标地市
4. 未达标地市使用红色字体显示
5. 排行榜按数量降序正确排列
6. 排行榜中达标/未达标标签正确显示
