## 1. 开发环境

- JDK 1.8
- Spring Boot 2.x
- Vue.js 2.x
- Element UI

**Dashboard.vue** - 数据加载

```javascript
async loadData() {
  this.loading = true
  try {
    const res = await getDashboardOverview({ month: this.selectedMonth })
    if (res.code === 200) {
      this.dashboardData = res.data
    }
  } finally {
    this.loading = false
  }
}
```

## 4. 配置说明

### 4.1 达标标准配置

在 `DashboardServiceImpl.java` 中定义：

```java
private static final int DEMAND_STANDARD = 6;   // 需求提单达标标准
private static final int REVIEW_STANDARD = 2;   // 评审参与达标标准
private static final int TRAINING_STANDARD = 8; // 培训上报达标标准
```

如需调整标准值，修改这些常量即可。

## 5. 扩展性设计

1. **达标标准可配置化**：可扩展为从配置文件或数据库读取
2. **缓存优化**：可引入Redis缓存统计数据
3. **数据导出**：可扩展Excel导出功能
4. **趋势对比**：可扩展月度趋势对比功能

## 6. 注意事项

1. 月份格式：前端使用 yyyyMM，培训上报表使用 yyyy-MM，需在Service层转换
2. participants字段为JSON数组格式，需使用ObjectMapper解析
3. 未达标地市列表按数量降序排列，便于查看差距最小的地市
