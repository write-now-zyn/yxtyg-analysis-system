# 一线体验官专项数据分析系统 - 设计文档

## 1. 系统架构

### 1.1 整体架构
```
┌─────────────────────────────────────────────────────────────┐
│                         前端层                               │
│                   Vue2 + ElementUI                          │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                         后端层                               │
│              SpringBoot 2.x + MyBatis Plus                  │
└─────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
┌─────────────────────────┐     ┌─────────────────────────┐
│        MySQL 8.0        │     │         Redis           │
│      (主数据库)          │     │       (缓存)            │
└─────────────────────────┘     └─────────────────────────┘
```

### 1.2 技术选型

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 前端框架 | Vue | 2.6.x | 渐进式JavaScript框架 |
| UI组件库 | ElementUI | 2.15.x | 饿了么UI组件库 |
| 后端框架 | SpringBoot | 2.7.x | 快速开发框架 |
| ORM框架 | MyBatis Plus | 3.5.x | 增强版MyBatis |
| 数据库 | MySQL | 8.0 | 关系型数据库 |
| 缓存 | Redis | - | 内存数据库 |
| Excel处理 | EasyExcel | 3.1.x | 阿里Excel处理工具 |

### 2.2 表结构设计

#### 2.2.1 体验官表 (t_experimenter)
| 字段名 | 类型 | 是否必填 | 说明 |
|-------|------|---------|------|
| id | bigint | 是 | 主键ID，自增 |
| city | varchar(20) | 是 | 地市 |
| name | varchar(50) | 是 | 姓名 |
| phone | varchar(20) | 是 | 电话 |
| email | varchar(100) | 否 | 邮箱 |
| role | varchar(50) | 否 | 角色 |
| remark | varchar(500) | 否 | 备注 |
| is_contact | tinyint | 否 | 是否接口人(0-否,1-是) |
| create_time | datetime | 是 | 创建时间 |
| update_time | datetime | 是 | 更新时间 |

#### 2.2.2 工单主表 (t_work_order)
| 字段名 | 类型 | 是否必填 | 说明 |
|-------|------|---------|------|
| id | bigint | 是 | 主键ID，自增 |
| order_no | varchar(50) | 是 | BOMC流水号 |
| declare_phone | varchar(20) | 否 | 申告电话 |

| handler_name | varchar(50) | 否 | 处理人 |
| city | varchar(50) | 否 | 申告地市 |
| create_time_str | varchar(50) | 否 | 创建时间（原始字符串） |
| initiator_name | varchar(50) | 否 | 发起人姓名 |
| handle_satisfaction_score | varchar(20) | 否 | 处理满意度评分 |
| dispatch_satisfaction_score | varchar(20) | 否 | 派单满意度评分 |
| unsatisfied_reason | varchar(500) | 否 | 不满意原因 |
| status | varchar(50) | 否 | 状态 |
| solution_history_str | text | 否 | 解决方案历史原始字符串 |
| content | text | 否 | 工单内容 |
| created_at | datetime | 是 | 记录创建时间 |
| updated_at | datetime | 是 | 记录更新时间 |

#### 2.2.4 评审会议纪要表 (t_review_meeting)

## 3. 接口设计

| /api/experimenter/{id} | GET | 获取体验官详情 |
